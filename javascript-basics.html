Lesson 1: What is JavaScript?

JavaScript is a programming language that makes websites interactive. It runs in your web browser and can:

  • Respond to user actions (clicks, typing, etc.)
  • Change what you see on the page
  • Store and manipulate data
  • Communicate with servers

Lesson 2: Variables (Storing Data)

Variables are like boxes that store data. You can create them with let, const, or var.

Best Practice: Use const by default, let when you need to change values.

// Creating variables const name = "Alice"; // Text (String) let age = 25; // Number let isStudent = true; // Boolean (true/false) let hobbies = ["reading", "coding", "gaming"]; // Array (list) console.log(name); // Output: Alice console.log(age); // Output: 25

Lesson 3: Data Types

JavaScript has different types of data:

// String - text const message = "Hello, World!"; // Number - integers and decimals const count = 42; const price = 19.99; // Boolean - true or false const isComplete = true; // Array - list of items const colors = ["red", "green", "blue"]; console.log(colors[0]); // Output: red // Object - collection of properties const person = { name: "Bob", age: 30, city: "New York" }; console.log(person.name); // Output: Bob

Lesson 4: Math and Operators

You can perform calculations and comparisons:

// Arithmetic const x = 10; const y = 3; console.log(x + y); // Addition: 13 console.log(x - y); // Subtraction: 7 console.log(x * y); // Multiplication: 30 console.log(x / y); // Division: 3.33... console.log(x % y); // Remainder: 1 // Comparison console.log(5 > 3); // true console.log(5 === 5); // true (equal value AND type) console.log(5 !== "5"); // true (not equal) console.log(10 >= 10); // true

Lesson 5: Conditionals (Making Decisions)

Conditionals let your code make decisions using if, else if, and else:

const age = 20; if (age >= 18) { console.log("You are an adult"); } else if (age >= 13) { console.log("You are a teenager"); } else { console.log("You are a child"); } // Output: You are an adult // Combining conditions const score = 85; const attendance = 95; if (score >= 80 && attendance >= 90) { console.log("Great job! A+ for you!"); } else if (score >= 70 || attendance >= 80) { console.log("Good work!"); } else { console.log("Keep trying!"); }

Lesson 6: Loops (Repeating Code)

Loops repeat code multiple times. Common loops are for and while:

// For loop - repeat a fixed number of times for (let i = 1; i <= 5; i++) { console.log("Count: " + i); } // Output: Count: 1, Count: 2, ..., Count: 5 // Loop through an array const fruits = ["apple", "banana", "orange"]; for (let i = 0; i < fruits.length; i++) { console.log(fruits[i]); } // For...of loop (cleaner way) for (const fruit of fruits) { console.log(fruit); } // While loop - repeat while condition is true let x = 0; while (x < 3) { console.log("x is " + x); x++; }

Lesson 7: Functions (Reusable Code)

Functions let you package code to reuse later:

// Simple function function greet(name) { return "Hello, " + name + "!"; } console.log(greet("Alice")); // Output: Hello, Alice! // Function with multiple parameters function add(a, b) { return a + b; } console.log(add(5, 3)); // Output: 8 // Arrow function (modern syntax) const multiply = (x, y) => x * y; console.log(multiply(4, 5)); // Output: 20 // Function without return function celebrate() { console.log("ðŸŽ‰ Congratulations! ðŸŽ‰"); } celebrate();

Lesson 8: Interacting with Web Pages (DOM)

JavaScript can change what appears on the page using the DOM (Document Object Model):

// Get an element by ID const heading = document.getElementById("myHeading"); // Change text heading.textContent = "New Title"; // Change style heading.style.color = "blue"; heading.style.fontSize = "24px"; // Add/Remove classes heading.classList.add("highlight"); // Create and add new elements const newButton = document.createElement("button"); newButton.textContent = "Click Me"; document.body.appendChild(newButton); // Listen for clicks newButton.addEventListener("click", function() { console.log("Button clicked!"); });
